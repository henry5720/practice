# Vite & React

## 專案環境建置
1. git clone 專案
``` bash
git clone https://github.com/henry5720/practice.git
```
2. 移至該專案路徑
``` bash
cd ./practice/react-app
```
3. install node.js 前往 [Node.js 官方網站](https://nodejs.org/en) 下載 Node.js 的安裝包
``` bash
node -v
```
4. install yarn 這個命令會全局安裝 Yarn
``` bash
npm install -g yarn
```
5. yarn 還原依賴
``` bash
yarn install
```
6. 執行dev模式
``` bash
yarn dev
```
